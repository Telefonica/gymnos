New experiment¶

In this tutorial we will create a new experiment for image classification using Trainer as the trainer and Dogs vs Cats as the dataset.

The name of the experiment will be my_experiment.

First of all, we will run the command gymnos-create:

gymnos-create experiment my_experiment

This will create the file conf/experiment/my_experiment.yaml.

It would look something like this:

conf/experiment/my_experiment.yaml¶
 1# @package _global_
 2# TODO: description about experiment
 3
 4defaults:
 5    - override /trainer: <trainer_name>  # TODO: set name of trainer to use
 6    - override /dataset: <dataset_name>  # TODO: set name of dataset to use
 7
 8trainer:
 9    <param>: <value>   # TODO: override default trainer params
10
11dataset:
12    <param>: <value>  # TODO: override default dataset params

First of all, we will add a description for our experiment:

2# Classify dogs vs cats using Transfer EfficientNet model.

Now, we will override the trainer with Transfer Efficientnet:

5- override /trainer: vision.image_classification.transfer_efficientnet

Then, we will override the dataset with Dogs vs Cats

6- override /dataset: dogs_vs_cats

Now, we can override any Trainer parameter:

 8trainer:
 9    classes: [dog, cat]
10    batch_size: 64

We can also override any Dogs vs Cats parameter:

11dataset:
12    max_workers: 8
13    force_download: false

Once you’re done modifying the experiment, you can run it using the gymnos-train command:

gymnos-train +experiment=my_experiment
