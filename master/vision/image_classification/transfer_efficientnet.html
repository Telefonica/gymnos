Transfer Efficientnet¶

Image classifier based on Transfer Learning with EfficientNet as backbone

pip install gymnos[vision.image_classification.transfer_efficientnet]

  • Trainer

  • Predictor

Trainer¶

gymnos-train trainer=vision.image_classification.transfer_efficientnet
class TransferEfficientNetTrainer(classes: List[str], num_workers: int = 0, batch_size: int = 32, num_epochs: int = 30, gpus: int = - 1, train_split: float = 0.6, val_split: float = 0.2, test_split: float = 0.2, accelerator: Optional[str] = None)¶

The expected structure is one directory for each class.

Let’s say we have classes=["dog", "cat"], then we should have two directories "dog" and "cat" containing the images:

dog/
    img1.png
    img2.png
    ...
cat/
    img1.png
    img2.png
    ...
Parameters

  • classes (List[str]) – Classes to train on, each class should be the name of the folder

  • num_workers (int) – Num workers to load data. If 0, loading data will be synchronous. If -1, all CPUs will be used.

  • batch_size (int) – Batch size for training and testing

  • num_epochs (int) – Number of epochs to train
