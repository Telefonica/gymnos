Installation¶

Prerequisites¶

Gymnos requires Python 3.7+.

Stable release¶

pip install git+https://github.com/Telefonica/gymnos.git@0.2

Bleeding-edge version¶

pip install git+https://github.com/Telefonica/gymnos.git@master --upgrade

Install from source¶

git clone https://github.com/Telefonica/gymnos.git
cd gymnos
pip install -e .

Development¶

We provide a Pipfile with all dependencies resolved for the project (including dev dependencies). First obtain Pipenv and then install Gymnos with development dependencies:

git clone https://github.com/Telefonica/gymnos.git
cd gymnos
pipenv install --dev

Refer to Pipenv documentation for more information.
