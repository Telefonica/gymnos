gymnos-upload¶

gymnos-upload [OPTIONS] MLFLOW_RUN_ID

Options

--name <name>¶

Name for SOFIA model

--description <description>¶

Description for SOFIA model

--public¶

Whether or not SOFIA model is public

Arguments

MLFLOW_RUN_ID¶

Required argument
